数据接口

1. 查询某日演出
接口格式：http://y.saoju.net/yyj/api/search_day/?date={}
例：http://y.saoju.net/yyj/api/search_day/?date=2023-07-25
返回json数据：
"date": "2023-07-25"
"show_list":
"city": "X城市"
"musical": "XX音乐剧"
"theatre": "XX剧院"
"time": "19:30"
"cast":
{"role": "A角色", "artist": "X演员"}
{"role": "B角色", "artist": "Y演员"}
2. 查询某日某城市演出
接口格式：http://y.saoju.net/yyj/api/search_day/?date={}&city={}
例：http://y.saoju.net/yyj/api/search_day/?date=2023-07-25&city=上海
返回json数据：
"date": "2023-07-25"
"city": "上海"
"show_list":
"musical": "XX音乐剧"
"theatre": "XX剧院"
"time": "19:30"
"cast":
{"role": "A角色", "artist": "X演员"}
{"role": "B角色", "artist": "Y演员"}
